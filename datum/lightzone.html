lightzone
/datum/lightzone

Lightzones are simple presets to determine what color and brightness the sky will be at a certain time. A list of lightzones must add up to 24 regarding start and end times.

Vars

end_hourThe HOUR at which this lightzone should end
light_alphaThe light power of this lightzone
light_colorThe light color of this lightzone
nameThe name of this lightzone
start_hourThe HOUR at which this lightzone should start
